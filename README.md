# 🔧 Reflex — Reparación de Celulares & Accesorios

Sitio web estático para **Reflex**, negocio de reparación de celulares, tablets, computadoras y venta de accesorios.

## Estructura

```
WEB/
├── index.html              # Página principal
├── src/
│   ├── styles/style.css    # Todos los estilos
│   ├── js/
│   │   ├── supabase.js     # Conexión a Supabase
│   │   ├── cart.js         # Carrito de compras (localStorage)
│   │   ├── main.js         # Carga de productos y renderizado
│   │   └── app.js          # Interfaz (navbar, menú, animaciones)
│   └── img/                # Imágenes y logo
```

## Usar

Abrí `index.html` en tu navegador o servilo con:

```bash
python -m http.server 8000
# o
npx http-server
```
